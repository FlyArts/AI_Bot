# numa_bot
The bot of FlyArts
# How to use
### 呼び出すだけ
```numa_bot```
### 挨拶をする
```こんにちは```
### 慰めてもらう
```ぬまきゅん、なぐさめて```
### 告白してみる
```ぬまきゅんすき```
### 時間を聞く
```今の時間```
### 天気予報を聞く
天気の情報取得はOpenWeatherMapのAPIを使っています。
今の所天気が取れる場所は仙台、東京、さいたま、熊谷です。
#### ※例
```東京の天気予報```
